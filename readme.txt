### Tira-Teima ###

Este é um software educacional para auxílio ao ensino de disciplinas introdutórias de computação. Viabiliza a simulação gráfica de um programa em execução, permitindo que o aluno veja uma representação do que acontece no computador ao longo da execução de um programa.
