Bem vindo ao Ajuda do Tira Teima.

O Tira Teima é uma ferramenta que visa auxiliá-lo no entendimento da execução de programas. Apesar de ser intuitivo, algumas dúvidas podem surgir de início. Em cada uma das abas superiores é possível aprender um pouco sobre uma das partes que compõem o Tira Teima. Logo abaixo temos a tela principal que você provavelmente já conhece.


Ela é composta por várias partes tais como:

  • código
  • mostrador
  • arquivos
  • console
  • barra de ferramentas
  • comentários
  • redimensionar

Código

Na parte esquerda do programa é localizado o código referente ao programa que será "executado". As cores seguem um padrão que facilita a leitura do mesmo, como se estivesse num editor de código fonte.

Não é possível editá-lo, mas é permitido copiar algum ou todo o trecho de código (seleção da parte do código com o mouse e pressionamento as teclas control + C). Com isso você pode tentar executar o código em qualquer compilador que desejar.


Mostrador

O Mostrador situa-se na parte superior direita do Tira Teima. É nela que será simulada a criação de variáveis na memória.

Cada variável possui um nome que a identifica (especificado no código). Elas também possuem um certo tamanho e cor, variando de programa a programa.


Arquivos

Os arquivos, quando o programa faz uso deles na execução do código, são visualizados na área central direita do Tira Teima.

Caso mais de um arquivo seja aberto, eles serão postos um do lado do outro, podendo ser abertos clicando-se na aba referente a cada um.


Console

O Tira Teima possui uma área que simula as entradas e saída de um Console. Seu funcionamento é automático em resposta à execução do programa em questão

O Console fica localizado na parte inferior direita do Tira Teima.


Barra de Ferramentas

Na parte superior do Tira Teima encontra-se a Barra de Ferramentas. Através dela é possível interagir com a ferramenta.

Ela é dividida em três partes:

  • Navegação
    • Reiniciar: permite iniciar a execução do código desde o início novamente
    • Anterior: retorna um passo na execução
    • Iniciar/Próximo: inicia e depois avança um passo na execução
    • Saltar Para Linha: Salta para uma linha específica. Caso a linha escolhida não tenha execução prevista, o Tira Teima saltará para a próxima linha executável
    • Campo para a linha: Local para indicar o número da linha. É possível apertar a tecla Enter ao invés de clicar no botão de saltar
  • Zoom
    • -: diminui o zoom no mostrador
    • Reniciar Zoom: ajusta o zoom para o nível inicial
    • +: aumenta o zoom no mostrador
  • Outros botões
    • Som: ativa ou desativa o som
    • Ajuda(?): abre a página de ajuda

Comentários

A parte de comentários só é visível quando algum trecho específico do código possui um detalhe no qual o professor deseja chamar atenção. Quando visível, os comentários são localizados na parte inferior e direita do Tira Teima.


Redimensionamento

É possível redimensionar as diversas áreas do Tira Teima (código, mostrador, console, arquivos, comentários) posicionando o cursor nas linhas (em vermelho nas figuras) e movê-las segurando o botão esquerdo do mouse.
